# Newsy
A RESTful API that shows top news stories to members

## Technologies used
- Python
- Flask
  - Flask-SQLAlchemy
- AI
  - Google colab
  - Tensorflow Keras
  - BERT model
  - Kaggle's BBC News Classification Dataset

## External Dependencies
- newspaper3k
