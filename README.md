# Newsy
A RESTful API that shows top news stories to members

## Technologies used
- Python
- Flask
  - Flask-SQLAlchemy
  - Flaks-Cors
  - Flask-Bcrypt
- AI
  - Google colab
  - Tensorflow Keras
  - BERT model
  - Kaggle's BBC News Classification Dataset
